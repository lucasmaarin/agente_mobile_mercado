Politica de Privacidade

Ultima atualizacao: 13 de fevereiro de 2026

Esta Politica de Privacidade descreve como o Mobile Mercado ("nos", "nosso" ou "aplicacao") coleta, usa e protege as informacoes dos usuarios e estabelecimentos que utilizam nossa plataforma de atendimento automatizado via WhatsApp.

1. Informacoes que coletamos

Coletamos as seguintes informacoes:

  • Dados de cadastro: nome, e-mail e informacoes de autenticacao via Firebase
  • Dados do estabelecimento: nome da empresa, configuracoes do agente, produtos e precos
  • Dados do WhatsApp Business: numero de telefone, identificadores da conta (phone_number_id, business_account_id) e tokens de acesso
  • Mensagens: conteudo das conversas entre clientes e o agente automatizado para processamento de pedidos
  • Dados de pedidos: itens, enderecos de entrega, valores e status dos pedidos

2. Como usamos as informacoes

Utilizamos as informacoes coletadas para:

  • Fornecer o servico de atendimento automatizado via WhatsApp
  • Processar e gerenciar pedidos dos clientes
  • Conectar o estabelecimento a API do WhatsApp Business
  • Melhorar a experiencia do usuario e o desempenho do agente
  • Gerar estatisticas de uso para o painel do estabelecimento

3. Compartilhamento de dados

Nao vendemos ou compartilhamos dados pessoais com terceiros, exceto:

  • Meta/WhatsApp: para envio e recebimento de mensagens via WhatsApp Business Cloud API
  • OpenAI: conteudo das mensagens e enviado para processamento pelo modelo de inteligencia artificial, sem identificacao pessoal do remetente
  • Firebase/Google: armazenamento seguro dos dados na infraestrutura do Google Cloud

4. Armazenamento e seguranca

Os dados sao armazenados no Firebase (Google Cloud Platform) com as seguintes medidas de seguranca:

  • Tokens de acesso armazenados de forma segura no banco de dados
  • Comunicacao via HTTPS em todas as conexoes
  • Autenticacao obrigatoria para acesso ao painel administrativo
  • Sessoes com expiracao automatica

5. Direitos do usuario

Voce tem direito a:

  • Acessar seus dados pessoais pelo painel do estabelecimento
  • Corrigir informacoes incorretas nas configuracoes
  • Excluir sua conta e dados associados
  • Desconectar o WhatsApp a qualquer momento pelo painel
  • Desativar o agente automatizado a qualquer momento

6. Retencao de dados

Mantemos os dados enquanto a conta estiver ativa. Ao excluir a conta ou desconectar o WhatsApp, os dados relacionados sao removidos. Historicos de conversas podem ser retidos por ate 30 dias para fins operacionais.

7. Uso da API do WhatsApp

Nossa aplicacao utiliza a WhatsApp Business Cloud API da Meta. Ao conectar seu numero, voce concorda com os Termos de Servico do WhatsApp Business e a Politica de Privacidade do WhatsApp.

8. Alteracoes nesta politica

Podemos atualizar esta politica periodicamente. Alteracoes significativas serao comunicadas pelo painel ou por e-mail.
