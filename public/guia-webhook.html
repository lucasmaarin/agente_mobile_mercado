Guia de Configuracao do Webhook WhatsApp

Este guia explica como configurar o webhook e a API no painel do Meta for Developers para que o bot de atendimento automatico funcione corretamente. Siga os passos abaixo com cuidado.

Parte 1 — Configuracao da API (numero e token)
1

Acessar o painel do Meta

Abra o navegador e acesse:

developers.facebook.com

Faca login com a conta Meta que tem acesso ao app Mobile Mercado Agente.

2

Abrir o app

No canto superior esquerdo, clique no menu Meus Apps e selecione Mobile Mercado Agente.

3

Ir para Configuracao da API

No menu lateral esquerdo, clique em WhatsApp e depois em Configuracao da API.

4

Selecionar o numero de telefone

Na secao "Enviar e receber mensagens", clique no dropdown De e selecione o numero de telefone do estabelecimento que deseja ativar.

Apos selecionar, anote o Phone Number ID que aparece abaixo do numero — voce vai precisar dele.

5

Gerar o Access Token

Logo abaixo do numero, clique no botao Gerar token de acesso.

Atencao: este token expira apos 24 horas. Para uso em producao, um token permanente sera gerado automaticamente pelo sistema.

Copie o token gerado (comeca com EAA...) — voce vai precisar dele no proximo passo.

6

Anotar o Business Account ID

No mesmo painel, procure a secao "Conta do WhatsApp Business". O Business Account ID aparece listado ali.

Anote esse numero tambem.

7

Cadastrar no sistema

Acesse o painel do Mobile Mercado Agente:

https://mobilemercado-agente.onrender.com/onboarding

Preencha os campos com os dados coletados:

  • Access Token — o token gerado (EAA...)
  • Phone Number ID — o ID do numero
  • Business Account ID — o ID da conta business
  • Numero de telefone — ex: +5511999999999

Clique em Conectar WhatsApp.

Parte 2 — Configuracao do Webhook
8

Ir para a configuracao do WhatsApp

No menu lateral esquerdo, clique em WhatsApp e depois em Configuracao.

9

Editar o Webhook

Role a pagina ate encontrar a secao Webhook. Clique no botao Editar.

Preencha os campos exatamente como abaixo:

URL de callback:
https://mobilemercado-agente.onrender.com/webhook
Token de verificacao:
zerograu_webhook_token

Clique em Verificar e salvar.

10

Assinar o campo "messages"

Apos salvar, clique no botao Gerenciar ao lado de Webhook.

Vai aparecer uma lista de campos. Encontre o campo messages e clique em Assinar.

Importante: sem assinar o campo "messages", o bot nao recebera as mensagens dos clientes e nao vai funcionar.
11

Pronto!

O webhook esta configurado. O bot ja pode receber e responder mensagens automaticamente pelo WhatsApp.

Para testar, envie uma mensagem para o numero conectado e o bot deve responder em alguns segundos.
